👋 Hi, I'm Abdullateef, a final year Biochemistry student building at the intersection of Machine Learning, AI, and Computational Biology.

My focus is on designing and applying AI and machine learning systems to solve problems that matter in biology and medicine, protein function prediction, multi-omics data analysis, transcriptomic pattern discovery, and the development of computational tools that work in resource-limited environments, not just on expensive GPU clusters.


🧠 Machine Learning & Deep Learning



I work primarily in PyTorch and TensorFlow, building deep learning systems from the ground up rather than just plugging data into existing frameworks. My experience includes designing multi-layer perceptron ensembles, handling extreme class imbalance through custom focal loss functions and dynamic class weighting, automated threshold calibration pipelines, and hierarchical inference architectures that cut computational cost without sacrificing prediction quality. I care about models that are not just accurate on paper but efficient and deployable in the real 
world.


🧬 Computational Biology & Bioinformatics 


I am comfortable across the full bioinformatics pipeline. On the omics side, my work spans transcriptomics (differential gene expression, RNA-seq analysis), metabolomics (LC-MS data interpretation), proteomics (protein feature engineering and function annotation), and survival analysis in clinical genomics datasets. I use GO and KEGG pathway enrichment to move from a list of genes or proteins to a biological story. I work in both Python and R depending on what the problem demands, and I understand the statistical reasoning behind the tools i use, not just how to run them.


⚙️ Backend, APIs & Deployment


Building a model is only half the work. I use FastAPI to build production-ready REST APIs that serve ML models to real users, handling everything from single-input queries to batch file uploads, memory-efficient model loading, and clean endpoint design. I think about deployment from the start of a project, not as an afterthought.


🖥️ Frontend Development


I build responsive, user-facing interfaces in React and JavaScript so that the computational tools i create are accessible to biologists and clinicians who do not live in the terminal. A good tool that no one can use is not a good tool.


🔬 Wet Laboratory Foundation


I am not purely a computational person. I have hands-on clinical laboratory training which means I understand the biological context behind omics data, where it comes from, what can go wrong during collection, and why certain patterns appear in the numbers.
